## Основные правила архитектуры и требования сервиса

- При любом запросе учитывай требования с низу

- Все правила архитектуры описаны в @docs/arch-rules.md

- Все требования к сервису описаны в @docs/requirements.md

- При изменении проекта нужно вносить описании архитектуры в документ @docs/architecture.md и использовать его в работе

- После каждого этапа проверяй собирается ли проект npm run build && npm run lint
